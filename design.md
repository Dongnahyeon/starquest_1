# StarQuest - 별자리 성취 관리 앱 디자인 계획

## 앱 개요

별자리 테마의 성취 목록 관리 앱. 사용자의 성취 항목은 하늘의 별로 표현되며, 완료할수록 별이 밝아진다. AI가 자동으로 성취 항목을 카테고리(별자리)로 분류한다.

## 화면 목록

### 1. 인트로 화면 (SplashIntro)
- **경로**: 앱 최초 진입 시 자동 표시 (AsyncStorage로 첫 진입 여부 판단 X, 매번 표시)
- **내용**: 밤하늘 배경에 별들이 하나씩 깜빡이며 나타나는 애니메이션
- **기능**: 2~3초 후 자동으로 홈 화면으로 전환
- **레이아웃**: 전체 화면 검정 배경, 무작위 위치에 별 50개 애니메이션, 중앙에 앱 이름 "StarQuest" 페이드인

### 2. 홈 화면 (Home / Constellation View)
- **경로**: `app/(tabs)/index.tsx`
- **내용**: 별자리 형태로 카테고리별 성취 그룹을 시각화
  - 각 카테고리는 별자리 이름으로 표시 (운동, 독서, 학습 등)
  - 카테고리 내 성취 항목들이 별로 표시되고 선으로 연결
  - 완료된 별은 밝게, 미완료는 어둡게 표시
- **기능**: 카테고리 탭 전환, 전체 진행률 표시
- **레이아웃**: 밤하늘 배경, 상단에 전체 진행률 바, 중앙에 별자리 SVG 뷰

### 3. 성취 목록 화면 (Achievement List)
- **경로**: `app/(tabs)/list.tsx`
- **내용**: 카테고리별로 그룹화된 성취 항목 목록
  - 각 항목 옆에 별 아이콘 (밝기로 완료 상태 표시)
  - 완료 체크박스
  - 완료 횟수 표시 (별이 몇 번 빛났는지)
- **기능**: 항목 체크/언체크, 항목 삭제, 카테고리 필터
- **레이아웃**: FlatList, 카테고리 헤더, 각 항목 카드

### 4. 성취 추가 화면 (Add Achievement)
- **경로**: `app/add.tsx` (모달 시트)
- **내용**: 성취 항목 이름 입력, AI 분류 결과 미리보기
- **기능**: 텍스트 입력 → AI 자동 카테고리 분류 → 확인 후 저장
- **레이아웃**: 바텀 시트 스타일, 입력창, AI 분류 결과 뱃지, 저장 버튼

### 5. 성취 상세 화면 (Achievement Detail)
- **경로**: `app/detail/[id].tsx`
- **내용**: 성취 항목 상세 정보, 완료 이력, 별 밝기 시각화
- **기능**: 완료 기록, 완료 취소, 삭제
- **레이아웃**: 중앙에 큰 별 애니메이션, 하단에 완료 이력 목록

## 핵심 사용자 플로우

### 성취 항목 추가 플로우
1. 홈 또는 목록 화면에서 `+` 버튼 탭
2. 성취 이름 입력 (예: "5km 달리기")
3. AI가 자동으로 카테고리 분류 (예: "운동")
4. 사용자 확인 후 저장
5. 해당 카테고리 별자리에 새 별 추가

### 성취 완료 체크 플로우
1. 목록 화면에서 항목 체크 또는 상세 화면에서 완료 버튼 탭
2. 별이 밝아지는 애니메이션 재생
3. 햅틱 피드백 (성공)
4. 완료 횟수 증가

## 별 밝기 시스템

| 완료 횟수 | 별 상태 | 색상 | 크기 |
|-----------|---------|------|------|
| 0 | 미완성 | `#4A5568` (어두운 회색) | 기본 |
| 1~2 | 희미한 별 | `#F6AD55` (주황) | 기본 + 약한 glow |
| 3~5 | 빛나는 별 | `#F5C842` (황금) | 기본 + 중간 glow |
| 6+ | 초신성 | `#FFF176` (밝은 노랑) | 기본 + 강한 glow |
